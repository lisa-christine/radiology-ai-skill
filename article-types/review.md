# Review Article

## Limits
| Element | Limit |
|---------|-------|
| Words | 3000 (Intro–Conclusion) |
| Abstract | 100-200 (unstructured) |
| References | 50 |
| Figures | 6 |
| Tables | 4 |

**Required**: Summary statement (single sentence), Essentials (3-5 bullets with page numbers)

**CLAIM**: Not required (no original AI development)

---

## Purpose
Comprehensive, in-depth reviews at level of general radiologist. Synthesize existing literature without new research.

**Note**: Systematic reviews and meta-analyses → submit as Original Research

---

## Structure

### Abstract (Unstructured)
Single paragraph: scope, key themes, main conclusions.

### Essentials (3-5 bullets)
Single-sentence takeaways with page references.

*Example*:
- Deep learning achieves radiologist-level performance for multiple chest findings (page 3).
- External validation remains limited in published studies (page 5).
- Implementation challenges include workflow integration and clinician trust (page 8).

### Body
Flexible structure based on topic:
- **Chronological**: Historical development
- **Thematic**: Organized by subtopics
- **Methodological**: Organized by approaches

**Suggested sections**:
1. Introduction (scope, organization)
2. [Theme 1]
3. [Theme 2]
4. [Theme 3]
5. Current Challenges
6. Future Directions
7. Conclusion

### Key Features
- Heavy citation density throughout
- Educational focus
- Balanced presentation
- No Methods/Results sections
