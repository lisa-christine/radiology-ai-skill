# AI in Brief

## Limits
| Element | Limit |
|---------|-------|
| Words | 1500 |
| Abstract | 200 (unstructured) |
| References | 20 |
| Figures + Tables | 4 total |

**Required**: Summary statement (≤255 chars), Key Points (≤3), CLAIM checklist

---

## Purpose
Original research applying previously described AI techniques. May be smaller scale than Original Research.

**Appropriate for**:
- Application of established methods to new problem
- Validation of existing algorithms
- Smaller-scale hypothesis-driven studies

---

## Structure

### Title
Same format as Original Research.

### Abstract (Unstructured, single paragraph)
Cover: purpose, methods (design, n, source), primary result with CI, conclusion.

### Introduction (1 paragraph)
Brief context, gap, purpose.

### Materials and Methods
Same required elements as Original Research, but condensed. Reference prior work for established methods rather than re-describing.

### Results
- Brief demographics
- Primary outcome with CI
- Key secondary findings

### Discussion (1-2 paragraphs)
- Summary
- Brief context
- Limitations
- Conclusion

---

## Key Differences from Original Research
- Unstructured abstract
- Combined figure/table limit (4 total)
- Can reference existing methods without full description
- More focused scope
