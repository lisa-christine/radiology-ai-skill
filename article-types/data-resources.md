## Purpose
Describe and link to databases valuable for AI research.

## Critical Requirement
Data **must** be deposited in approved public repository. NOT acceptable: personal websites, lab servers, "available upon request."

**Approved repositories**: TCIA, PhysioNet, Zenodo, Figshare, OpenNeuro, Synapse, Hugging Face Datasets

---

## Structure

### Summary Statement
*Template*: "[Dataset] provides [n] [data type] with [annotations] for [application], enabling [research opportunity]."

### Introduction
- Clinical/research motivation
- Gap in available resources
- Purpose

### Dataset Description
- Source and collection process
- Population characteristics
- Imaging parameters
- Size and composition

### Annotation Methodology
- Reference standard definition
- Annotation process
- Quality control
- Inter-annotator agreement

### Data Access
- Repository URL
- Access conditions
- Data use agreement
- File format, naming conventions

### Potential Applications
- Intended use cases
- Benchmark tasks

### Discussion
- Limitations
- Comparison to existing datasets
- Planned extensions
